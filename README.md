# ZomieksTikTok

**Tech Stack:**

- **Frontend:** Next.js 15 (App Router), React 19, TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** Cloudflare Workers (Edge Runtime), D1 (SQLite), KV, R2, Durable Objects
- **Payments:** OZOW (Instant EFT), PayFast (Card Payments)
- **Email:** Resend (Edge-compatible transactional email)
- **Database ORM:** Drizzle ORM

Zomieks is a tiktok styled app that allows users to post, comment, like and share short form videos and earn per view.
